# 标签页收集器

一个基于 Plasmo 框架的浏览器插件，用于一键收集和管理浏览器标签页。

## 功能

- 🔍 一键收集当前窗口所有标签页
- 📋 清晰展示所有收集的链接
- 🚀 一键打开所有收集的链接
- 👆 单击打开单个链接
- 🗑️ 清空收集列表

## 开发

```bash
npm run dev
```

## 构建

```bash
npm run build
```

构建完成后，在 `build/chrome-mv3-dev` 或 `build/chrome-mv3-prod` 目录中找到插件文件。

## 安装

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `build/chrome-mv3-dev` 目录

## 使用

1. 点击浏览器工具栏中的插件图标
2. 点击"收集所有标签页"按钮
3. 查看收集的标签页列表
4. 点击"打开所有链接"或单击某个链接打开
